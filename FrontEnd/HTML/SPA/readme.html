项目主要内容

  1. 水质分析:对三维水质数据（时间，地点，监测项）， 进行两个维度的报表分析（表格，图形）
  2. 水质实时数据:以地图为基础，获取对应地点的数据，与水质标准做简略比较
  3. 待机主题动画:纯CSS实现
  4. 登录，注册页面:默认账户： abc@qq.com 密码：123456
  5. 数据上传:允许用户上传数据，格式要求 .cvs (仅实现前端部分)
  6. 自定义水质标准
